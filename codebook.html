district_id

INTEGER HARMONIZED

District (Zila)

LADM: LA_AdministrativeSource

Description

Identifies the administrative district (Zila) where the land parcel is located. Punjab is divided into 36 districts, each with unique administrative boundaries and land record systems. This variable is harmonized across all datasets and time periods to ensure consistent geographic analysis.

Availability

Dataset Years Notes
Registry Transactions 2015-2024 All records
Land Ownership Current All records
Mutation Trends 2010-2024 All records

Universe

All land parcels and transactions in Punjab province. No records are excluded (NIU = 0).

Important

  • 2017 Boundary Changes: Toba Tek Singh (code 37) was created from parts of Faisalabad and Jhang districts. Pre-2017 records have been recoded to reflect current boundaries.
  • CLRMIS vs ESORS: Both systems use the same district coding scheme. No harmonization required.
