Geographic Standardization

Understanding administrative boundary changes and geographic crosswalks in Punjab land data

Overview

Punjab's administrative boundaries have changed significantly over time. Districts have been split, tehsils reorganized, and sub-registrar jurisdictions modified. PLAP provides standardized geographic identifiers and crosswalks to enable consistent time-series analysis despite these boundary changes.

Punjab Administrative Structure

Punjab uses a hierarchical administrative structure for land management. Understanding this hierarchy is essential for accurate data analysis.

About Geographic Crosswalks

Crosswalk tables map geographic units between different time periods. They allow you to aggregate or disaggregate data to maintain consistent boundaries across time series analysis.

How Crosswalks Work

When a district is split, the crosswalk provides allocation weights based on mauza-level data to distribute historical aggregates to new boundaries, or combine new data to match old boundaries.

Available Crosswalk Tables
